Книга наряд-заданий

1. Описание проекта
Электронная книга наряд-заданий предприятия ведет учет всех выполненных, текущих и планируемых работ,
а также предоставляет отчетность в различных формах

Реализованы 3 роли: Рабочий (Рабочий персонал), Начальник участка (Руководство), Администратор

Рабочий после авторизации видит только свои наряды: В работе, Завершенные

Начальник участка имеет возможность:
 - Редактировать справочник оборудования (добавлять, редактировать, удалять оборудование)
 - Выдавать наряд по участку на смену, редактировать наряд, завершать наряд (после проверки выполнения)
 - Просмотреть все наряды на выбранную дату / по определенному участку

Администратор имеет все функции начальника участка и:
 - Редактировать справочник сотрудников (добавлять, редактировать, удалять)

employees (сотрудники), orders (наряды), myauth (авторизация)

2. Запуск проекта
cd ordersbook
pip install -r requirements.txt
python manage.py runserver

Авторизация по ссылке: http://127.0.0.1:8000/login/
